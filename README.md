# 🧠 Memory Match Challenge  
An interactive web-based card matching game built with **HTML, CSS, and JavaScript**.  

## 🎮 About the Game  
Memory Match Challenge is a fun browser game that tests your memory skills.  
- Enter your name  
- Choose a difficulty (Easy, Medium, Hard)  
- Match all pairs in the shortest time with the fewest moves  

Your stats are saved in the **local leaderboard**, so you can track your progress and compete for the top spot.  

## ✨ Features  
- 🎴 Dynamic card grid with smooth 3D flip animations  
- ⏱ Timer and move counter  
- 🏆 Local leaderboard (per difficulty)  
- 📱 Responsive design (desktop & mobile)  
- 🎉 Win screen with game stats  

## 🚀 Play Online  
👉 [Play the game here](https://samarbal.github.io/Memory-Game/)  

## 🛠️ Tech Stack  
- **Frontend:** HTML, CSS3, Vanilla JavaScript  
- **Storage:** LocalStorage (for scores & leaderboard)  
- **Hosting:** GitHub Pages
